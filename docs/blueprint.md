# **App Name**: MentorVerse

## Core Features:

- Expert Discovery: Browse a curated list of industry experts, filtering by skill, industry, and experience level, inspired by Convo.vip.
- Session Booking: Schedule and manage 1-on-1 mentorship sessions with experts, mirroring the simple booking flow of Convo.vip.
- Personalized Learning Paths: Access structured learning paths tailored to specific skills and career goals.
- Task Management: Track progress on assigned tasks within learning paths, with clear status updates (Pending, Submitted, Reviewed) from AntCode inspiration.
- AI Task Tool Review: Mentors leverage AI to analyze submitted tasks, identify knowledge gaps, and provide feedback using the provided tool.
- Mentor Dashboard: A dedicated space for mentors to manage assigned students, review tasks, and schedule sessions.
- Role-Based Authentication: A clear role based process when signing up as either a student or a mentor

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for professionalism and intellect.
- Background color: Light grey (#F5F5F5) for landing pages, dark grey (#333333) for dashboards.
- Accent color: Electric blue (#03A9F4) for CTAs and progress states, to provide a visual pop.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel.
- Use consistent and clear icons across the platform for navigation and task management.
- Landing pages will feature a clean layout with generous white space. The dashboards will utilize a card-based layout for content organization.
